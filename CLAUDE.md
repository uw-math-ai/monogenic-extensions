## Project Overview

This is a Lean 4 formalization project for the [UW Math AI Winter 2026 program](https://sites.math.washington.edu/ai/winter2026.html) on algebraic geometry.

**Goal**: Formalize Lemmas 3.1 and 3.2 from [arXiv:2503.07846](https://arxiv.org/abs/2503.07846) - proving that an injective, finite, étale ring homomorphism between local rings is monogenic (i.e., the target ring is generated by a single element over the source). (Completed, now we have no sorries). We want to add our results to Mathlib in a PR, so our focus is to clean up our code, simplify it, and get it Mathlib ready (in structure and in style).

### Key Formalization Choices

The code uses Mathlib's `IsAdjoinRootMonic` rather than custom definitions. The conclusion `∃ f : R[X], Nonempty (IsAdjoinRootMonic S f)` packages both:
- S ≅ R[X]/(f) as an R-algebra
- f is monic

The derivative unit condition is a separate theorem (`isUnit_aeval_derivative_minpoly_of_adjoin_eq_top`).

### Proof Strategy (Lemma 3.2, fully étale case, in Generator.lean)

1. **Primitive element theorem**: `Field.exists_primitive_element` gives generator β₀ over residue field
2. **Lift β₀ to β in S**: via `Ideal.Quotient.mk_surjective`
3. **Nakayama's lemma**: `Algebra.adjoin R {β} = ⊤` (using `map_maximalIdeal` for étale)
4. **Construct isomorphism** via `IsAdjoinRoot.mkOfAdjoinEqTop'`: Cayley-Hamilton bounds degree, Orzech property gives injectivity
5. **Derivative is a unit**: separability of residue field extension + lift

### Proof Strategy (Lemma 3.1, partial étale case, in MonogenicOfNonEtale.lean)

1. If R → S is already étale, apply Lemma 3.2 directly
2. Otherwise, apply Lemma 3.2 to the étale quotient R/p → S/q to get B₀, lift to B
3. **Case 1**: If aeval B f₁ already generates the maximal ideal modulo m_R·S, use `adjoin_eq_top_of_quotient` (Nakayama)
4. **Case 2**: Otherwise, adjust B to B' = B + q₀ (where q₀ generates q). Taylor expansion shows aeval B' f₁ = q₀ · (unit), so the adjusted generator works

## Mathematical Background

### Lemmas from the Paper

**Lemma 3.2** (étale case): Let (R, m_R) and (S, m_S) be local rings with R → S finite, injective, and étale. Then:
1. There exists β ∈ S with R[β] = S (`exists_adjoin_eq_top`)
2. f(z) ∈ R[z], β's minimal polynomial, is monic and f'(β) ∈ S× (`isUnit_aeval_derivative_minpoly_of_adjoin_eq_top`)
3. S ≅ R[X]/(f) as R-algebras (`IsAdjoinRoot.mkOfAdjoinEqTop'`)

**Converse** (`IsAdjoinRootMonic.algebra_etale`): If S ≅ R[X]/(f) with f monic and f'(root) a unit, then R → S is étale.

**Lemma 3.1** (partial étale case): Let (R, m_R) and (S, m_S) be local integral domains with R integrally closed and S a UFD. If R → S is finite and injective, and there exists a height 1 prime q ⊂ S such that R/(q ∩ R) → S/q is étale, then there exists a monic f with S ≅ R[X]/(f) (`exists_isAdjoinRootMonic_of_quotientMap_etale`).

### Connection to Stacks Project

The notion of monogenic extension is related to **standard étale** maps (Tag 00UB): R→S is standard étale if S ≅ R[x]_g/(f) where f is monic and f' is a unit. Every étale map is locally standard étale (Tag 00UE). The converse theorem `IsAdjoinRootMonic.algebra_etale` constructs a `StandardEtalePair` directly.
